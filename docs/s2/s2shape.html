    Module s2.s2shape

    Represents polygonal geometry in a flexible way.

    Classes

    NameDescription
    S2Shape The purpose of S2Shape is to represent polygonal geometry in a flexible way. It is organized as a collection of edges that optionally defines an interior. All geometry represented by an S2Shape must have the same dimension, which means that an S2Shape can represent either a set of points, a set of polylines, or a set of polygons.
