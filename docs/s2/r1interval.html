    Module s2.r1interval

    Represents a closed, bounded interval on the real line.

    Structs

    NameDescription
    R1Interval An R1Interval represents a closed, bounded interval on the real line. It is capable of representing the empty interval (containing no points) and zero-length intervals (containing a single point).
